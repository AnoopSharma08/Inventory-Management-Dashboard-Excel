Inventory Management Dashboard (Excel + Power Pivot)
Project Summary

This project showcases an interactive Inventory Management Dashboard built using Microsoft Excel, Power Query, and Power Pivot to monitor stock movement, track inventory levels, and support data-driven decision-making.

The dashboard highlights key metrics such as total stock received (4,540 units), dispatched (2,827 units), and current inventory (1,713 units), along with an overall stock availability of 38%. It also provides monthly trend analysis to identify patterns in inventory flow.

A key feature is the identification of low-stock items through a Bottom 10 report based on reorder levels, helping prevent stockouts. Users can dynamically filter data by store and product for detailed insights.

Key Features

Tracks inventory movement (Received vs Dispatched vs Available)

Monthly trend analysis for better planning

Low-stock identification based on reorder levels

Interactive filtering by store and product

KPI indicators for quick performance overview

Tools Used

Excel

Power Query

Power Pivot

Business Value

Improves inventory visibility, reduces manual tracking, and enables better replenishment and operational decisions.
